# NBAClient
- The client for NBA Stats

## Requirements

- npm or yarn installed
- node.js installed
## Installation

Download the example [or clone the repo](https://github.com/paterlisia/NBAClient):

<!-- #default-branch-switch -->

```sh
git clone https://github.com/paterlisia/NBAClient.git
cd NBAClient
```

Install it and run:

```sh
npm install
npm start
```

## Deploy

```sh
npm run build
```
